# Perubahan v0.4.0 - 2026-04-08

## Ringkasan
Rilis fitur besar: green screen (chroma key) untuk video overlay, perbaikan proporsi overlay, pemisahan mode single/multi source, deteksi video Facebook privat, dan peningkatan kompatibilitas video.

## Added
- **Green Screen (Chroma Key)** pada Video Overlay
  - Checkbox untuk mengaktifkan chroma key per overlay
  - Color picker untuk memilih warna key (default hijau `#00ff00`)
  - Slider Similarity (0.1–1.0) untuk rentang warna yang dihapus
  - Slider Blend (0.0–1.0) untuk kelembutan tepi transparansi
  - FFmpeg filter: `format=yuva420p` → `chromakey` untuk transparansi yang benar
  - Preview phone menampilkan ikon dan border khusus untuk overlay green screen
- **Deteksi video Facebook privat/friends-only**
  - Facebook Reels yang bersifat privat mengembalikan error deskriptif: "Video ini tidak dapat diakses. Kemungkinan video bersifat privat atau hanya untuk teman (friends-only)."
  - Deteksi berdasarkan title generik "Facebook" dan ukuran HTML < 60KB

## Changed
- **Pemisahan mode Single Source vs Multi Source**
  - 1 video: tampil File Info panel + slider "Clone Count" langsung mengatur jumlah clone
  - 2+ video: tampil Source Videos list + slider berubah jadi "Default Clones per Source"
  - Menghilangkan tumpang tindih antara panel info dan source list
- **Overlay Size slider** diperluas dari max 50% menjadi max 100%
- **Proporsi video overlay** diperbaiki — ukuran dihitung dari lebar video utama dengan menjaga aspect ratio asli overlay menggunakan ffprobe
- **HTML unescape** diterapkan pada seluruh teks sebelum regex pencarian URL video Facebook (bukan hanya pada URL yang ditemukan)
- Membersihkan elemen HTML duplikat (Resolution/Codec/Bitrate) yang muncul ganda di UI

## Fixed
- **Video overlay gepeng** — sebelumnya menggunakan `scale2ref` yang salah menghitung tinggi. Sekarang dimensi dihitung eksak di Python via ffprobe: `target_w = main_w × pct`, `target_h = target_w × ov_h / ov_w`
- **Green screen tidak transparan** — filter `format=yuva420p` harus diterapkan SEBELUM `chromakey`, bukan sesudahnya, agar ada alpha channel untuk ditulis
