# Perubahan v0.5.0 - 2026-05-05

## Ringkasan
Menambahkan dukungan download video dari **Shopee Video** (termasuk short link `id.shp.ee/...` / `shp.ee/...` dan link share `sv.shopee.<region>/share-video/...`). Shopee kini muncul sebagai opsi platform di dropdown Single Link, Batch Links (auto-detect), Background Music link, dan Video Overlay link.

## Perubahan
- `src/api.py`
  - Tambah `download_from_shopee(url)`, `get_shopee_download_progress()`, dan helper internal `_shopee_download(url, download_dir)`.
  - Resolusi short URL `shp.ee` → universal-link → `sv.shopee.<region>/share-video/...` (urllib auto-redirect, dengan fallback manual jika landing di halaman `universal-link` desktop).
  - Ekstraksi metadata video dari blok `__NEXT_DATA__` di HTML share page (Next.js SSR), pakai `props.pageProps.mediaInfo.video.watermarkVideoUrl`.
  - Nama file output: `shopee_<caption>.mp4` (atau `shopee_<postId>.mp4` jika caption kosong).
  - Reuse oleh `extract_bgm_from_link` dengan menambahkan `'shopee'` ke `download_methods`.
- `src/web/index.html`
  - Tambah `<option value="shopee">Shopee</option>` di `#selectSource` (Single Link) dan `#selectBgmSource` (BGM dari link).
- `src/web/app.js`
  - Tambah entri `shopee` di `SOURCE_CONFIG` (label, placeholder, API method, progress method).
  - Tambah `shopee: 'Shopee'` di label dict BGM.
  - Tambah deteksi URL Shopee di `detectSourceFromUrl()` — mendukung domain `shp.ee` dan `*.shopee.*` (semua region: `.co.id`, `.com.my`, `.sg`, dll.).
  - Tambah `<option value="shopee">Shopee</option>` di template kartu Video Overlay.

## Dampak
- User: dapat menempel link Shopee Video langsung (single, batch, atau sebagai sumber BGM/Video Overlay) dan mendapatkan file MP4 untuk di-clone seperti platform lain.
- Catatan: Shopee hanya mengekspos URL berwatermark publik di share page. Video hasil download akan mengandung watermark Shopee. Untuk menyembunyikan/menutup watermark, user dapat memanfaatkan kombinasi efek `crop`/`blur`/`zoom` atau Text/Video Overlay yang sudah ada.
