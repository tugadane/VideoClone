# Perubahan v0.3.0 - 2026-03-10

## Ringkasan
- Implementasi frontend lengkap: konversi UI Preview v2 menjadi aplikasi fungsional dengan semua interaksi, logic, dan integrasi ke backend Python via pywebview API.

## Perubahan

### File Baru
- `src/web/index.html` — UI fungsional lengkap, dikonversi dari `ui_preview_v2.html`.
  - **Custom Title Bar**: drag region, logo Clone Studio, badge versi, tombol minimize/maximize/close.
  - **Panel Kiri — Task Queue**:
    - Progress bar global (persen, label, estimasi sisa waktu).
    - Daftar clone item (waiting → processing → done) dengan scroll otomatis ke item aktif.
    - FFmpeg status indicator (hijau Ready / merah Not Found).
  - **Panel Kanan — Clone View**:
    - Drop zone untuk drag & drop video (visual feedback).
    - Unified link download area: dropdown pilihan sumber (TikTok, Instagram Reels, Facebook Reels, YouTube Shorts, Google Drive) + input URL + tombol download + progress bar download.
    - File info card: filename, size, duration, resolution, codec, bitrate.
    - File Configuration: naming template (dengan tag buttons), output format selector, output folder browser.
    - Clone Engine: slider jumlah clone (1–100), method selector, estimasi ukuran & waktu, tombol START/CANCEL/RESET.
  - **Panel Kanan — History View**: toggle via icon di top bar, daftar riwayat clone dengan detail, tombol Clear All.
  - **Guide & Settings Modal**: quick guide 3 langkah, perbandingan clone method, konfigurasi FFmpeg path, default values, toggle notifikasi pop-up & suara, tombol Save Settings.
  - **Completion Modal**: dialog sukses clone dengan jumlah & waktu total, tombol Open Folder.
  - **Styling**: Tailwind CSS, Work Sans font, Material Symbols icons, theme hitam + oranye, custom scrollbar, gradient glow effects, smooth transitions.

- `src/web/app.js` — Frontend logic lengkap (~725 baris).
  - **Initialization**: `init()` — load config, check FFmpeg, load history, setup event listeners.
  - **Event Listeners**: semua interaksi UI (window controls, tab switching, file selection, link download, folder browse, cloning, settings, drag & drop).
  - **File Selection**: `selectVideoFile()`, `onFileSelected()`, `clearFile()`, `resetAll()`.
  - **Link Download (Unified)**: `SOURCE_CONFIG` mapping untuk 5 platform, `downloadFromLink()` dengan progress polling, `updateLinkPlaceholder()`.
  - **Cloning Control**: `startCloning()`, `cancelCloning()`, polling progress setiap 500ms.
  - **Progress UI**: `updateProgress()`, `updateTaskList()` (render item done/processing/waiting), `onCloneComplete()`, `onCloneError()`.
  - **Estimates**: `updateEstimates()` — kalkulasi estimasi ukuran (MB/GB) dan waktu berdasarkan method.
  - **Template Preview**: `updateTemplatePreview()` — preview real-time nama file output.
  - **Tab Switching**: `toggleHistory()`, `showTab()`.
  - **History**: `loadHistory()`, `clearHistory()` — render daftar riwayat clone.
  - **FFmpeg Check**: `checkFFmpeg()` — tampilkan status Ready/Not Found.
  - **Config/Settings**: `loadConfig()`, `saveSettings()`, `browseFFmpeg()`.
  - **Toggle Switch**: `toggleSwitch()`, `setToggle()` — animasi toggle button.
  - **Utility**: `escapeHtml()` — sanitasi output HTML.

### File Pendukung Build
- `CloneStudio.spec` — Konfigurasi PyInstaller untuk build executable.
- `build.bat` — Script build otomatis.
- `start.bat` — Script untuk menjalankan aplikasi dari source.
- `installer.iss` — Inno Setup script untuk membuat Windows installer.

## Dampak
- Aplikasi Clone Studio sepenuhnya fungsional dan siap dipakai.
- Semua fitur yang didesain di v0.1.0 telah diimplementasikan.
- User bisa clone video dari file lokal maupun link online (5 platform).
- UI responsif dengan progress real-time, estimasi, dan notifikasi completion.
- Aplikasi bisa di-build menjadi executable dan installer Windows.
