# v0.7.1 — Tombol Back di Clone History

**Tanggal:** 2026-05-06
**Tipe:** patch (bug-fix UX)

## Ringkasan

Halaman Clone History sebelumnya tidak menampilkan tombol kembali yang eksplisit. Satu-satunya cara kembali ke main view adalah meng-klik ulang ikon clock di top bar — tidak intuitif. Sekarang ada tombol **Back** di pojok kiri atas halaman Clone History.

## Perubahan

### `src/web/index.html`
- Tombol baru `#btnBackToMain` di header Clone History, dengan ikon `arrow_back` + label "Back". Ditempatkan di kiri judul "Clone History".

### `src/web/app.js`
- Wiring `#btnBackToMain` ke `toggleHistory()` (handler yang sama dengan tombol clock di top bar — efeknya kembali ke `tab-clone` dan reset state ikon).

## Verifikasi

- `node --check src/web/app.js` → OK
- HTML structure tetap responsif: header di-flex jadi 2 kolom (`Back + judul` di kiri, `Clear All` di kanan); pada layar kecil tetap satu baris (`flex-1 min-w-0` mencegah overflow).
