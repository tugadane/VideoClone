# Perubahan v0.6.0 — 2026-05-05

## Ringkasan
Menambahkan fitur **Hide Watermark Region** untuk menutup/menghilangkan watermark statis pada video sumber sebelum proses cloning. Fitur ini dirancang untuk menyembunyikan watermark Shopee Video (kiri-tengah) yang tidak bisa dihindari di endpoint publik Shopee, namun didesain generic agar bisa dipakai untuk platform lain (TikTok, Instagram Reels, YouTube Shorts).

## Perubahan
- `src/cloner.py`
  - Tambah pre-processing pass: `_preprocess_watermark_all()` dipanggil di awal `_run()` sebelum loop clone, sehingga watermark dihilangkan satu kali per source (bukan per clone) — efisien.
  - `_resolve_watermark_regions()` menggabungkan region manual user dengan preset auto-Shopee (kiri-tengah, `delogo+blur`) saat `auto_shopee=true` dan source memiliki `source_platform == 'shopee'`.
  - `_apply_watermark_filters()` membangun perintah FFmpeg yang men-render file `wmrem_<id>.mp4` di temp dir. Mendukung 4 metode:
    - `delogo` — in-paint berbasis pixel tetangga (filter `delogo`).
    - `delogo_blur` — kombinasi `delogo` + `boxblur` selektif (rekomendasi, hasil paling natural).
    - `blur` — selektif `boxblur` saja (sensor look).
    - `cover` — `drawbox` solid translucent.
  - `_cleanup_preprocessed()` di blok `finally` agar temp file selalu dihapus walau cloning gagal/cancel.
- `src/api.py`
  - Setiap method `download_from_*` menambahkan `info['source_platform']` (`'tiktok'`, `'reels'`, `'fbreels'`, `'ytshorts'`, `'shopee'`, `'gdrive'`) supaya frontend bisa mendeteksi platform asal file.
- `src/web/index.html`
  - Panel baru **"Hide Watermark Region"** (collapsible) sejajar dengan Text Overlay / Video Overlay.
  - Toggle "Auto-hide Shopee watermark saat source dari Shopee" (default ON).
  - Tombol preset cepat: Shopee (kiri-tengah), TikTok atas/bawah, IG atas, YT Shorts bawah.
  - Tombol "Add Region" untuk region manual.
- `src/web/app.js`
  - State `WATERMARK_PRESETS`, fungsi `addWatermarkRegion(presetKey?)`, `getWatermarkHideOptions()`, `toggleWatermarkHidePanel()`.
  - Setiap region card berisi input X/Y/W/H persen, dropdown Method (4 opsi), slider Strength.
  - `startCloning()` mengirim `watermark_hide` di payload dan `source_platform` per source.
  - `renderSourceList()` menampilkan badge platform di setiap source (mis. "Shopee", "TikTok"), supaya user tahu mana yang akan kena auto-hide.
  - `resetAll()` membersihkan panel watermark (regions, toggle, counter).

## Dampak
- **User**: dapat secara otomatis menutup watermark Shopee tanpa konfigurasi (cukup download dari Shopee → start clone). Untuk platform lain, tinggal klik tombol preset atau definisikan region manual.
- **Performansi**: setiap source kena 1 pre-pass FFmpeg (~5–15 detik untuk video 1 menit). Hasil di-cache di temp file dan dipakai ulang untuk semua clone dari source itu.
- **Catatan**: tidak ada teknik post-processing yang 100% invisible. Method `delogo_blur` memberikan hasil paling natural untuk watermark teks pada background tekstur (sudah diuji pada link Shopee user). User dapat menggabungkan dengan efek `crop`/`zoom` untuk hasil lebih bersih bila perlu.
- **Reverse-compat**: payload `watermark_hide` opsional. Jika tidak dikirim atau `enabled:false`, perilaku clone sama persis dengan v0.5.0.
