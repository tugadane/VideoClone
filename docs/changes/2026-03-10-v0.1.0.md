# Perubahan v0.1.0 - 2026-03-10

## Ringkasan
- Phase awal proyek: diskusi spesifikasi, pembuatan UI preview, dan penetapan keputusan desain fundamental Clone Studio.

## Perubahan

### File Baru
- `ui_preview.html` — UI Preview v1 dengan layout sidebar kiri (icon-only) + konten kanan, tab Clone/History/Settings, custom CSS styling.
- `stitch_reference.html` — Referensi desain dari Google Stitch, mengadopsi Tailwind CSS, Work Sans font, Material Symbols, gradient glow.
- `ui_preview_v2.html` — UI Preview v2 (**APPROVED**). Perbaikan besar dari v1:
  - Sidebar menu dihapus (fitur tidak banyak, tidak perlu navigasi).
  - Task Queue dipindah ke panel kiri (permanen, selalu visible).
  - History sebagai icon toggle di top bar.
  - Guide & Settings digabung dalam modal popup.
  - Layout final: Top Bar → [Task Queue | Main Content].
- `docs/README.md` — Spesifikasi proyek lengkap (nama, stack, theme, fitur).
- `docs/CHANGELOG.md` — Catatan perubahan proyek.
- `docs/DEVELOPMENT.md` — Panduan teknis implementasi.

### Spesifikasi Proyek yang Ditetapkan
- Nama: Clone Studio
- Stack: Python 3.13 + pywebview + FFmpeg (portable)
- Theme: Black (#000000) + Orange (#ff5c00)
- Fitur inti: clone video, naming template, output config, progress, history, notifikasi

### Keputusan Desain
- Sidebar menu tidak diperlukan karena fitur tidak banyak.
- Task Queue selalu visible di panel kiri agar user bisa monitor progress.
- Settings & Guide digabung dalam 1 modal popup.
- History cukup sebagai icon toggle di top bar.
- Clone method: Fast (metadata + filter) vs Standard (light re-encode).
- Konflik nama file: auto-rename dengan suffix angka.
- Notifikasi: pop-up + suara (default ON).

## Dampak
- Fondasi proyek terbentuk: spesifikasi, arsitektur, dan desain UI sudah fix.
- UI Preview v2 menjadi acuan utama untuk implementasi frontend.
- Semua keputusan desain terdokumentasi sebagai referensi development selanjutnya.
